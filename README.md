# DRE-WEB

Projeto web estático simples contendo páginas e assets para o DRE.

## Descrição

Site estático com as páginas principais e os recursos em `DRE/` (CSS e JS). Ideal para hospedar em GitHub Pages ou outro servidor estático.

## Estrutura do projeto

- `Index.html` — página principal
- `login.html` — página de login
- `DRE/` — pasta de recursos
  - `assets/css/events.css` — estilos
  - `assets/js/events.js` — scripts

## Como executar

1. Clone este repositório.
2. Abra `Index.html` no seu navegador (`File -> Open` ou arraste o arquivo para o navegador).
3. Pode acessar atraves do link <link>https://rafael1102.github.io/DRE/<link>

Não há dependências ou build steps — é um site estático.
